# 网好卡菌 (NetGoodLagBacteria)

> “心跳漏了一拍？不，是你的屏幕漏了一帧。”

**网好卡菌** 是一款灵感来源于独立音乐游戏《节奏医生》（Rhythm Doctor）的屏幕特效与视觉干扰控制面板。就像《节奏医生》中那些试图破坏你节奏感的“病毒”一样，本程序会在你的屏幕上制造各种视觉故障、弹窗干扰和系统级“卡顿”错觉。

这是一款**安全、可控的恶作剧/视觉特效模拟器**。它不会破坏你的系统文件，也不会窃取你的数据，它唯一的目的就是让你的屏幕看起来像是感染了某种极具节奏感的赛博病毒。

---

## 🎵 核心特性：渐进式干扰系统

在《节奏医生》中，Boss 战的干扰往往是循序渐进的。**网好卡菌** 完美继承了这一设计理念：

所有的视觉特效都不会在开启的瞬间达到最大强度。程序内置了一个**全局强度曲线（Intensity Ramp）**，在启动后的前两分钟内，特效会非常轻微，几乎难以察觉；随着时间的推移（约 5 分钟后），干扰频率和强度会呈指数级上升，最终达到满屏故障的“高潮”阶段。

这种设计非常适合用于恶作剧——受害者一开始只会觉得“电脑好像有点卡”，直到屏幕彻底陷入疯狂。

## 🦠 病毒图鉴：内置特效列表

| **随机错误声音** | 播放 Windows 默认的错误提示音，配合视觉故障食用更佳。 |
| **屏幕闪烁** | 屏幕亮度或颜色突然闪烁，模拟显示器接触不良。 |
| **弹出消息框** | 弹出带有“网好卡菌”标题的嘲讽弹窗。弹窗会在屏幕上随机游走，且**绝对不会抢夺当前窗口的焦点**。 |

## 🎮 控制面板与操作指南

程序启动后，会显示一个带有亚克力（Acrylic）毛玻璃模糊效果的现代控制面板。

### 快捷键（硬核抢救机制）
为了防止你自己在测试时被“初见杀”，程序内置了全局快捷键：

- **`Shift + O`**：**一键暂停/恢复所有特效**。按下后，所有正在运行的特效会立即停止，屏幕会被清理干净，所有游走的弹窗也会瞬间被强制关闭。
- **`Shift + Delete`**：**一键高潮（Trigger All）**。按住此组合键，所有已勾选的特效会无视渐进式强度曲线，瞬间以最大频率爆发。

### 终极彩蛋：重启警告
在程序运行约 2 分钟后，系统会弹出一个极其逼真的“Windows 将在 5 分钟后重新启动”的警告框。
- 这个警告框同样会四处游走，且无法被轻易点击。
- 如果你试图点击“取消”，它会在 5 秒后自动消失，并随后弹出连续的嘲讽消息：“哈哈！你被我骗了！不会真的重启的啦~”
- 如果你真的点到了“确定”……它会执行真实的关机命令（带有 5 分钟倒计时），然后再次嘲讽你。
